# PRAMAANIK Demo Script (5-8 Minutes)

## Pre-Demo Setup
```bash
docker compose up --build   # ~3 min spin-up
pnpm run seed:demo          # seeds synthetic fraud data
```
Open http://localhost:3000 in browser.

---

## 0:00 – 0:45 — The Story (Emotional Hook)

> "In 2025, the CAG audited India's flagship skill development program — PMKVY.
> They found that 94.53% of 95.9 lakh beneficiary records had missing or invalid bank accounts.
> 12,122 bank accounts were shared across 52,381 supposedly distinct participants.
> ₹10,000+ crore of public money — and nobody caught it until the audit arrived two years late."

Show the CAG Report No. 20/2025 finding on screen.

> "We built PRAMAANIK to ensure this never happens again."

Switch to the landing page → scroll to architecture diagram.

> Every disbursement is recorded as an immutable event.

## 1:30 – 3:00 — Live Disbursement (Admin Role)

Click **Admin Portal** → show the dashboard with stats.

> "I'm logged in as a treasury official. Let me create a new disbursement under PM-KISAN."

> "Notice the oracle checks: GST valid ✓, bank account unique ✓, geotag verified ✓.
> The transaction is now on the Besu chain."

> "Now let me simulate the Gujarat Dahod MGNREGA pattern —
> I'll create 5 disbursements of ₹4.98 lakh each to the same vendor, just below the ₹5L threshold."

Submit 5 rapid transactions.
